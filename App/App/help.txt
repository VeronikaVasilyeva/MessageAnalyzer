Справка по коммандам TwitterBot.

help - вывод справки
clear - очистить консоль
logout - разлогиниться
exit - выход из программы

add - добавить логины аккаунтов для анализа. Для окончания ввода ввести пустую строку.
stat username - вывести статистику для username в консоль
post username - опубликовать от имени авторизованного пользователя статистику по username

Авторизация от имени пользователя доступна только через браузер!!!
